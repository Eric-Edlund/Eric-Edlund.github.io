The content of a webpage is stored in Hyper Text Markup Language (HTML). HTML code is stored in plain text files ending in .html.

Elements

An HTML document is a collection of Elements, each notated with an opening tag, like , and a matching closing tag, like . An element represents a thing.

There are many different types of elements (things) that you can have on your website (currently, this html document). What type of element you have is specified in the opening and closing tags. A , for example, could be a cat element, while could represent a dog.

Nested Elements

Elements may contain other elements in a parent-child relationship. This is done by putting an element between the opening and closing tags of another. One parent element (the enclosing one) may contain multiple children. Naturally, an element can only have one parent.

This cat element contains a heart, two lungs, and the defining characteristic of a cat. The cat's organs belong to it, and are considered siblings to each other.

Components of a Web Page

For an html file to be a proper website, there are a few important elements it needs to have. Firstly, needs to appear at the top of the document. This tells the browser what it's looking at. Under that, you need an html element.

will contain all the information about the web page. Its has two children:

  1. The head element contains data about the website: what language it's in, what should be displayed on the tab. Nothing in the head will be displayed on the page.
  2. The body contains the content on the page. Its children hold text, images, links, videos and whatever else your site has on it.

Inside the head, you can see the title element. title contains the text to be displayed on the site's tab (for this site, WebDev Tutorial). Note that in the text inside the element, Text Displayed on the Tab, spaces will be preserved. This is only true for single spaces: tabs, multiple spaces and newlines are trimmed.

In the body, there is only 1 element: a paragraph element, notated as just p. Paragraphs will format their text to make it look like a parargaph. p is, again, one of many many different elements in html you can use. Paragraph for example will format its text to have the normal font size and appear like a paragraph.

Helpful Tools

An IDE like Visual Studio Code is highly recommended as it will bring non-obvious issues in your HTML code to your attention. It will prevent you from spending an hour searching over your code trying to figure out why it won't work and then realizing it was because you forgot to put a ", or close an element, or some other insignificant detail you shouldn't spend time thinking about. Frequently, it will also be able to explain your mistake and offer a solution to fix it. Your IDE will be like a set of training wheels for now, and provide other powerful tools to you later.

There are a hundred IDEs out there, and just like spirit animals and wands, you have to find yours.

Opening your Web Page

Your webpage (currently just a file ending in .html) can be opened with a web browser like Chrome, Firefox, Safari, Edge, Brave or Opera. If you are reading this tutorial, you're on a web browser! Open your page by right clicking your HTML file and selecting Open with... and picking a browser.
